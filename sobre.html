HyperText

HTML5

HTML é a sigla de HyperText Markup Language, expressão inglesa que

significa "Linguagem de Marcação de Hipertexto". Consiste em uma

linguagem de marcação utilizada para produção de páginas na web, que

permite a criação de documentos que podem ser lidos em praticamente

qualquer tipo de computador e transmitidos pela internet.

HTML5

HTML é a linguagem de marcação padrão para páginas da Web.

Com HTML você pode criar seu próprio site.

HTML é fácil de aprender - você vai gostar!

Aprendizado fácil com HTML "Experimente você mesmo"

Com nosso editor "Experimente você mesmo", você pode editar o código HTML e visualizar o resultado:

Exemplo

< !DOCTYPE html >
< html >
< head >
< title > Page Title < /title >
< /head >
< body >
< h1 > Isso é um titulo < /h1 >
< p > Isso é um paragrafo. < p >
< /body >
< /html >
